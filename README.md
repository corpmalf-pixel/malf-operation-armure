# MALF Operation Armure

Site web professionnel premium pour l'activité de coaching sportif MALF Operation Armure.

## Stack
- Site statique HTML / CSS / JavaScript
- Structure compatible Vercel
- Liens relatifs

## Lancer en local
```bash
python3 -m http.server 8000
```
Puis ouvrir `http://localhost:8000`

## Déploiement Vercel
Le projet est prêt pour Vercel dès l'import du repository GitHub.

## Pages
- `/` Accueil
- `/about` À propos
- `/services` Services
- `/contact` Contact
