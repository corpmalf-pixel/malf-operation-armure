# Images du projet

Dépose ici les visuels du site pour qu'ils soient directement versionnés dans le repository.

## Structure recommandée
- `branding/` : logo, favicon, éléments d'identité
- `hero/` : visuels de couverture et image principale
- `about/` : photos liées à la présentation
- `services/` : images pour les offres et accompagnements
- `gallery/` : visuels additionnels, transformations, ambiance
- `testimonials/` : portraits ou visuels clients
- `social/` : images de partage OG si besoin

## Règles
- Utiliser des noms de fichiers simples, en minuscules, avec tirets.
- Formats recommandés : `.jpg`, `.png`, `.webp`, `.svg`
- Optimiser les images avant ajout.

## Exemple
- `hero/coach-training-session.webp`
- `branding/logo-malf-operation-armure.svg`
- `services/strength-program.jpg`
